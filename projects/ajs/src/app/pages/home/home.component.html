💫 Reactive State

Simple, granular, reactive state, made possible by independent reactive data structures that are based on native data structures, and extend RxJS' Observable.

✨ Extensive Features

Features like undo/redo, persistence, immutability and observable events come built in with ActiveJS' reactive data structures.

🚥 Streamline Async APIs

ActiveJS provides mechanisms that make streamlining and sharing asynchronous data APIs' state effortless.

🍭 Succinct & Simple

ActiveJS strives to enhance DX by reducing boilerplate, and limiting the amount of abstract utility concepts.
